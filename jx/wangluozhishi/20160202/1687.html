华为路由器ar1220丢失存储怎么办

修复方式请参考如下步骤，另AR1220-S最新版本为V200R003C01SPC300 + V200R003SPH008补丁，推荐升级到此主流维护版本。感谢您使用huawei的设备。

1）设备掉电重新启动，当设备启动到“Press Ctrl+B to break auto startup“阶段，快速同时按键盘上Ctrl和B两个按键。

BIOS Creation Date : Dec 31 2011, 17:33:44                                                                                         

DDR DRAM init : OK                                                                                                                 

Start Memory Test ? ('t' or 'T' is test):skip                                                                                      

Copying Data : Done                                                                                                                

Uncompressing : Done                                                                                                                

USB2 Host Stack Initialized.                                                                                                       

USB Hub Driver Initialized                                                                                                          

USBD  Wind River Systems, Inc. 562 Initialized                                                                                     

Octeon Host Controller Initialize......Done.                                                                                       

Press Ctrl+B to break auto startup ... 9

*注意Ctrl+B阶段只有10秒钟，请尽快按住相应的键盘，如果超时错过，需要重新掉电启动设备后再次等待输入。


2）输入Ctrl+B后，系统提示需要输入相应的密码,输入英文字母“huawei“即可.

Press Ctrl+B to break auto startup ... 9                                                                                            Enter Password:huawei

 

3）输入密码后，进行如下的菜单，输入数字5：

Main Menu                                                                                                                  

                                                                                                                                    

    1. Default Startup                                                                                                             

    2. Serial Menu                                                                                                                  

    3. Network Menu                                                                                                                

    4. Startup Select                                                                                                              

    5. File Manager                                                                                                                

    6. Reboot     

Enter your choice(1-6): 5

4）然后再输入数字1，查看flash内文件。

  File Menu                                                                                                                       

1. Flash file system                                                                                                                                                                                                                          

0. Return        

Enter your choice(1-2):1

 

5）再输入数字1，查看flash内的文件列表；

    Flash file system MENU                                                              1. List file in flash                                                  

2. Delete file in flash                                                    

3. Rename file in flash                                                     

4. Format Flash file system                                                

0. Return

Enter your choice(1-6): 1


6）系统会列出flash中所有的文件，及文件名称，这里需要查找末尾为“.CC“的文件，这里为AR1220_V200R001C00SPC500.cc 文件名,记录相应的文件名称。

Enter your choice(1-4):1                                                       

Directory of flash:/* :                                                         

-rw-  68229504  Sep 30 2011 11:04:12   AR1220_V200R001C00SPC500.cc               

-rw-         0  Nov 24 2011 18:40:54   web.zip                                 

-rw-       396  Oct 24 2011 00:43:33   rsa_host_key.efs                        

-rw-       540  Oct 24 2011 00:43:45   rsa_server_key.efs                               

-rw-      5717  Nov 22 2011 12:54:04   mon_file.txt                                       

-rw-      1665  Nov 07 2011 14:34:31   iascfg.zip                              

-rw-    524838  Nov 07 2011 14:34:42   private-data.txt                        

7 files found!                                                                               

*注：按照原先的名称大小写进行文件命名记录。

 

7）记录好需要记录的文件名，然后选择“0“返回到上一级菜单。

        Flash file system MENU                                                  

    1. List file in flash                                                      

    2. Delete file in flash                                                    

    3. Rename file in flash                                                     

    4. Format Flash file system                                                

    0. Return                                                                  

                                                                                

Enter your choice(1-4):0


8）然后再选择“0“返回到主菜单。

        File Menu

    1. Flash file system

    0. Return

Enter your choice(1-1):0

 

9）输入数字“4“，进入到启动设置菜单；

        Main Menu

    1. Default Startup

    2. Serial Menu

    3. Network Menu

    4. Startup Select

    5. File Manager

    6. Reboot

Enter your choice(1-6):4                                                        


10）再输入数字“1“，进入下一级中；

        Select Boot Device                                                      

    1. Flash                                                                    

    0. Return    

Enter your choice(1-1):1                                                        

NOTE: Boot file must be .cc or .CC                                             

                                                                                

Current boot file: flash:/AR1220_V200R001C00SPC500.cc                         

Press ENTER directly for no change.                                            

Or, please input the new file name: flash:/                                     


11）在光标停留处，输入之前在第6步记录的文件名称，输入完回车，并且在后面输入字母“y“进行确认。

Save load state word...OK!

NOTE: Boot file must be .cc or .CC

Current boot file:

Press ENTER directly for no change.

Or, please input the new file name: flash:/AR1220_V200R001C00SPC500.cc

Save the bootfile name: flash:/AR1220_V200R001C00SPC500.cc ? Yes or No(Y/N) y

Save load state word...OK!


注：在输入文件名称时在光标处不要有空格键，如果输入错误请不要使用backspace退格键，需要重新再次按照步骤11）进行输入。


12）输入完后，系统会自动返回到主菜单。然后再输入数字“1“即可。

Save load state word...OK!                                                     

        Main Menu                                                              

    1. Default Startup                                                         

    2. Serial Menu                                                              

    3. Network Menu                                                            

    4. Startup Select                                                          

    5. File Manager                                                             

    6. Reboot                                                                  

                                                                               

Enter your choice(1-6):   1


13）系统会重新的启动。

Enter your choice(1-6):1                                                       

Enter auto startup                                                             

                                                                               

Now boot from flash:/ AR1220_V200R001C00SPC500.cc, please wait....            

   

14）等系统重新启动完全后，输入下面的命令将故障前配置文件恢复，然后回车即可：

<Huawei> startup saved-configuration iascfg.zip                                 

This operation will take several minutes, please wait..........                

Info: Succeeded in setting the file for booting system                          

 

15）如果设备上使用的为AR1220_V200R001C00SPC200版本还需要同步激活补丁AR1220_V200R001SPH001.pat。

查看设备上的存储的补丁信息

<Huawei>cd flash:                                                              

<Huawei>dir *.pat                                                               

Directory of flash:/                                                           

                                                                               

  Idx  Attr     Size(Byte)  Date        Time(LMT)  FileName                     

    0  -rw-         45,184  Jan 12 2012 20:08:27   ar1220_v200r001sph001.pat   

                                                                               

217,168 KB total (145,888 KB free,70,928 KB used,192 KB unknown )              

激活AR1220_V200R001SPH001.pat补丁。

<Huawei>patch load AR1220_V200R001SPH001.pat all run                           

  This operation will take several minutes, please wait.........................

..................................................                              

  Patch operation succeeded                                                    


查看设备上的已激活的补丁信息

<Huawei>display patch-information                                              

  Patch version     :    ARV200R001SPH001                                       

  Patch package name:    flash:/ar1220_v200r001sph001.pat                      

<Huawei>display startup                                                        

MainBoard:                                                                      

  Startup system software:                   flash:/ar1220_v200r001c00spc200.cc 

  Next startup system software:              flash:/ar1220_v200r001c00spc200.cc 

  Backup system software for next startup:   null                              

  Startup saved-configuration file:          null                              

  Next startup saved-configuration file:     flash:/iascfg.zip                 

  Startup license file:                      null                              

  Next startup license file:                 null                              

  Startup patch package:                     flash:/ar1220_v200r001sph001.pat  

  Next startup patch package:                flash:/ar1220_v200r001sph001.pat  

  Startup voice-files:                       null                              

  Next startup voice-files:                  null

 

16）如上步骤完成后，再次重新等设备启动后，相关业务可正常恢复。
