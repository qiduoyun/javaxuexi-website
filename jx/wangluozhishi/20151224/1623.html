cisco路由器ISDN配置实例 蘑菇  发布于 2015-12-24 15:14:42

Cisco 路由器同时支持回拨功能，我们将路由器Router1 作为Callback Server,Router2 作为Callback
Client。 


Router1:
hostname router1
user router2 password cisco
!
isdn switch-type basic-net3
!
interface bri 0
ip address 192.200.10.1 255.255.255.0
encapsulation ppp
dialer map ip 192.200.10.2 name router2 572
dialer load-threshold 80
ppp multilink
dialer-group 1
ppp authentication chap
！
dialer-list 1 protocol ip permit
!
Router2:
hostname router2
user router1 password cisco
!
isdn switch-type basic-net3
!
interface bri 0
ip address 192.200.10.2 255.255.255.0
encapsulation ppp
dialer map ip 192.200.10.1 name router1 571
dialer load-threshold 80
ppp multilink
BRI0:
192.200.10.1
电话号码：
571
BRI0:
192.200.10.2
电话号码：
572
Router1 Router2
ISDN
23
dialer-group 1
ppp authentication chap
！
dialer-list 1 protocol ip permit
!
