json使用入门教程 炒菜  发布于 2015-11-14 18:41:09

json作为一种数据结构，被广泛用于各种数据交换的数据格式。使用json需要导入相应的jar包，在本例子的源码中有提供。

这里通过一个例子来介绍json的使用，将map数据转换成json对象，将json对象转换成字符串，然后输出。

1、导入jar包。

2、具体类：Json_Demo 

package com.javaxuexi.json;

import java.util.HashMap;
import java.util.Map;

import net.sf.json.JSONObject;

public class Json_Demo {
/**
 * @param map
 * @return 将map集合转换成json
 */
public JSONObject mapToJson(Map map){
    JSONObject jsonObject=new JSONObject();//创建json对象
    jsonObject.putAll(map);//将map转换成json对象
    return jsonObject;
}

/**
 * @param jsonObject
 * @return 将json对象转换成字符串
 */
public String jsonToString(JSONObject jsonObject){
    return jsonObject.toString();
}

    public static void main(String[] args) {
        // TODO Auto-generated method stub
        Json_Demo json_Demo=new Json_Demo();
        Map map=new HashMap();//创建map对象，并添加数据
        map.put("no-1", "张三");
        map.put("no-2", "李四");
        map.put("no-3", "王五");
        JSONObject jsonObject=json_Demo.mapToJson(map);//将map转换json
        String string=json_Demo.jsonToString(jsonObject);//json转string
        System.out.println(string);//输出json字符串
    }

}

 

3、运行查看结果：

我们可以看到，json数据结果和map非常相似，数据都是以键值对（key、value）进行存储的。
